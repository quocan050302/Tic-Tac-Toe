## Installation & Setup

### Prerequisites

Ensure you have **Yarn** installed. If not, install it first:

```sh
npm install -g yarn
```

### Clone the Repository

```sh
git clone https://github.com/quocan050302/Tic-Tac-Toe.git
cd tic-tac-toe
```

### Install Dependencies

```sh
yarn install
```

### Start the Development Server

```sh
yarn dev
```
